特别开心来到了结束语这个篇章。来到这个篇章，意味着你已经学完了这个课程，而我也已经写完了整个专栏。

Claude Code 的任何重要演进，我会给大家及时加餐或者通过直播来分享的，因此我在结束语部分就不说技术和课程的事儿，我只说一说从开课（1 月 28 日）到此刻（4 月 15 日）我的心路变迁。

“一日不见，如三秋兮”，这句话出自《诗经·王风·采葛》，以夸张手法表达对亲友的深切思念。在 AI 时代，反而变得不那么夸张。就这短短的几个月专栏创作期间，我们经历了 OpenClaw 从大流行到热度消退，Harness Engineering 概念从天而降，Claude Code 源码深夜泄露，以及 Hermes 取代 OpenClaw 引领新的潮流…

如果你一天没刷手机，第二天听到大家都在谈论全新的名词，而你却全然无知，你是否会觉得自己非常失落，甚至觉得是身处另一个世界。这样一幕你是否有点眼熟：“啊？！你居然没听说过 Hermes？？？”

这其实是一个非常荒谬的事情。

因此在这个被加速的年代，我们反而要把自己的心态稳定下来。大家一起思考一下，在这个时代什么东西其实并没有变。什么东西是真正值得的。

## 不变之一：世界的复杂性

这个世界其实从来都是复杂的，而这也是我们工程师存在的根本原因。

图片参考了我在 Datawhale 做的分享整理而来，如果你对我前阵子在 Datawhale 带来的 Agent Harness 分享感兴趣，可以查看 这个链接 。

我们工程师一直都是在跟系统的复杂度做斗争。技术架构演进二十几年不变的是什么？不变的就是如何驾驭复杂性。我们驾驭了对象，我们驾驭了系统架构，我们驾驭了分布式，我们驾驭了微服务… 我们所做的所有努力，都是希望能够降低复杂系统的复杂度。现在，轮到我们工程师去驾驭这个充满 AI 的崭新世界。

这个世界可以快速变化，但是我们仍然要用工程师的耐心、智慧和勇气，去勇敢的应战，并且通过抽象能力加上结构化，把很乱的东西里边的本质抓出来，让复杂的新世界变得更容易驾驭。

正如我的一个群友留言所说：“我们是从分布式、微服务，云计算和大数据时代存活过来的工程师，我不会觉得 Agent 时代的技术有什么了不起的。我们可以的。”—— 我非常同意这个看法。我们可以的。

## 不变之一：关系的可贵性

人和人之间的连接很珍贵。在 AI 到处都是的年代尤其如此。越是 AI，越呼唤人和人之间真正的接触和友谊，而技术可以是桥梁。

在课程上线之初，我就收获了大家大量的发自内心的支持，这种能量我感受到了，也形成了我的创作来源。

大厂资深工程师“6 点起床学习的和尚”在专栏中持续贡献高价值评论，得到了大家的点赞和认可。有学员留言“和尚实战经验丰富，极客时间可否邀请佳哥和和尚共创一集内容？”

在专栏临近结束，我最新的公开分享中，真的邀请到了和尚，和他详细取经，共创出 SDD 落地真实场景，奉献给大家。大家的愿望成真了。

![](https://static001.geekbang.org/resource/image/7f/8e/7fa01dafe682c948baa7bfe40ee19e8e.png?wh=721x462) ![](https://static001.geekbang.org/resource/image/7a/e4/7a5245e75dc9458191b8939f72c6a6e4.png?wh=1214x656)

技术之外，佳哥感受到的都是大家满满的学习热情，以及和尚全心全意在百忙的工作中（10 点才下班，6 点又要起床）的浓浓友情。

有这些东西支撑着，我们就没那么害怕 AI 了。

## 时代加速变化，我们如何前行？

好，再来说说我们应该如何前行。因为时代的加速变化是真实存在的，任何人都要做好应对，何况是走在技术前沿的我们。简单来说，我觉得这三点更重要了。

### 思考深度

一定要把问题想深，想透。要把思考当乐趣。你的思考深度和刨根问底的精神将成为你和他人的分水岭。AI 时代你多了一个思考的对手和战友，你要利用好这个战友。它是个威胁，同时也是最好的礼物。

![](https://static001.geekbang.org/resource/image/7c/9b/7c0fd664dabda8ayy1feb35650328e9b.png?wh=715x392)

### 学习速度

学而时习之，不亦说乎。爱学习的人在 AI 时代有福了。因为你拥有了一个最好的助手和学习搭子。学习的成本也变得很低。如果你不喜欢学习，我很难想象你将以什么样的感受与你后续的几十年人生同处。

我真心希望更多人能够喜欢上大脑神经元更新的感觉，就像大模型的训练过程一样，越训练越强，越训练越快。神经元越改变，你从这个世界上获取的信息越真实（很多我们原本以为是真实的东西并不是真的）。

学习的过程其实是改变自己旧观念的过程，对于所有的旧观念，我们要舍得抛弃，舍得改变。这需要勇气，因为每个人似乎都很执着于“旧”的自己。我不知道为什么。我更喜欢改变后的、更新的自己。

### 抱团取暖

我们真的很需要携手同行。佳哥不是什么大师，我只是一个愿意带着大家一起学习的人。我所有的群（最早的已经 7、8 年了），从来没冷清过。很多群都很快凉了，佳哥的群越来越热闹。大家讨论最新技术、彼此介绍工作机会，甚至交友，征婚（真的）。

因为群友说，只要是欣赏佳哥的，肯定爱学习，肯定正能量。—— 我深以为然。（广告：佳哥的群逐渐都满了。如果你有需要，除了课程群，你还可以通过极客时间最近开设的 行动营 跟进我的思考动态，我的新专栏 《Agent 设计模式之美》 与我共同研究 Agent 设计语言，或者搜索“咖哥 AI”付费支持佳哥，加入佳哥的星球。）
